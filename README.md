Zoom Clone

Zoom Clone App that works on Android

Features:

  * Google Sign In
  * Create New Meeting
  * Join Meeting
  * Mute/ UnMute Audio
  * Turn on/off video cam
  * Change Display name while joining
  * Raise Hand
  * Start screen sharing
  * Mute Everyone
  * Chat
  * End Call
  * See your meeting history
  * Log Out

Tech Used:
  
 Server-
  * Firestore
  * FirebaseAuth
  * Jitsi Meet
  * Google Sign In
  
 Client-
  * Flutter
